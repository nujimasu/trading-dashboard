================================================================
  Trading Dashboard — データ更新手順ガイド
  （エンジニア以外の方向け）
================================================================

このファイルでは、ダッシュボードのデータを最新の状態に保つための
手順を、2種類の更新パターン別に説明します。

----------------------------------------------------------------
  はじめに：「ターミナル」を開く方法
----------------------------------------------------------------

Mac の場合：
  1. キーボードで「Command + スペース」を押す
  2. 「ターミナル」と入力してEnterを押す
  3. 黒い画面（ターミナル）が開く

開いたら、まず以下のコマンドを1回だけ入力してEnterを押してください。
（ダッシュボードのフォルダに移動する操作です）

  cd /Users/junusami/Documents/Claudeプライベート/trading-dashboard

※ 以降の操作はすべてこのフォルダで行います。


================================================================
  【初回のみ】必要なソフトをインストールする
================================================================

以下のコマンドをターミナルに貼り付けてEnterを押してください。
1〜2分かかります。一度やれば次回以降は不要です。

  pip3 install -r requirements.txt

「Successfully installed」という文字が出れば成功です。


================================================================
  パターン①：週1回の「フルデータ更新」
  （推奨タイミング：毎週月曜の朝・市場開始前）
================================================================

この更新では、全銘柄のデータをゼロから取得し直します。
約 15〜30分 かかります（銘柄数によります）。

◆ 手順

  1. ターミナルを開き、以下を入力してEnter：
     cd /Users/junusami/Documents/Claudeプライベート/trading-dashboard

  2. 以下のコマンドを入力してEnter：
     python3 pipeline/run_pipeline.py

  3. 画面に進捗が表示されます。
     「Pipeline complete」と表示されたら完了です。

  4. ブラウザで以下を開いてダッシュボードを確認：
     http://localhost:8000
     ※ ダッシュボードが起動していない場合は、
       先に「ダッシュボードの起動方法」を実施してください。

◆ 注意事項
  ・インターネット接続が必要です
  ・途中でエラーが出ても、最後まで自動的に進みます
  ・「No stocks passed Stage 3 filter」と表示されても正常です
    （相場が弱い日はフィルタを通る銘柄がゼロになることがあります）


================================================================
  パターン②：毎日の「軽量データ更新」
  （推奨タイミング：平日の市場前後、1〜2分で完了）
================================================================

週次推奨銘柄のその日の価格・RR・ブレイクアウト状況を更新します。
週次更新を週1回行ったうえで、毎日これを実行するのが理想です。

◆ 手順

  1. ターミナルを開き、以下を入力してEnter：
     cd /Users/junusami/Documents/Claudeプライベート/trading-dashboard

  2. 以下のコマンドを入力してEnter：
     python3 pipeline/run_pipeline.py --daily-only

  3. 「Daily picks updated」と表示されたら完了です。


================================================================
  ダッシュボードの起動方法
================================================================

データ更新とは別に、ダッシュボードの画面を開く手順です。

  1. ターミナルを開き、以下を入力してEnter：
     cd /Users/junusami/Documents/Claudeプライベート/trading-dashboard

  2. 以下のコマンドを入力してEnter：
     python3 run.py

  3. 「Uvicorn running on http://127.0.0.1:8000」と表示されたら、
     ブラウザで以下のURLを開いてください：
     http://localhost:8000

◆ ダッシュボードを閉じるには
  ターミナル上で「Control + C」を押してください。

◆ 注意
  ダッシュボードを表示中は、ターミナルを閉じないでください。
  閉じると画面が見られなくなります。


================================================================
  APIキーの設定状況
================================================================

このダッシュボードでは3つの外部データサービスを使用します。
現在はすべて設定済みです（変更が必要な場合の手順も記載）。

┌─────────────────┬────────────────────────────────┬────────────┐
│ サービス          │ 用途                            │ 設定状況   │
├─────────────────┼────────────────────────────────┼────────────┤
│ FMP              │ 米国株ファンダメンタルデータ       │ 設定済み   │
│ FRED             │ 米経済指標（CPI・金利・雇用等）    │ 設定済み   │
│ Polygon.io       │ 市場ニュース                     │ 設定済み   │
└─────────────────┴────────────────────────────────┴────────────┘

◆ キーを変更・更新したい場合

  1. ターミナルで以下を入力してEnter：
     open ~/.zshrc

  2. テキストエディタが開くので、以下の行を探して新しいキーに書き換え：
     export FMP_API_KEY=（FMPのキー）
     export FRED_API_KEY=（FREDのキー）
     export POLYGON_API_KEY=（Polygonのキー）

  3. 保存後、ターミナルで以下を実行：
     source ~/.zshrc

◆ 各サービスのキー取得先（再取得が必要になった場合）

  FMP      : https://financialmodelingprep.com/register（無料・250回/日）
  FRED     : https://fred.stlouisfed.org/docs/api/api_key.html（無料・無制限）
  Polygon  : https://polygon.io/dashboard（無料プランは5回/分制限）

◆ ニュース・経済指標の更新

  ニュースや経済指標は週次パイプラインには含まれていません。
  以下のコマンドで単独実行できます（1〜2分）：

    cd /Users/junusami/Documents/Claudeプライベート/trading-dashboard
    python3 pipeline/news_collector.py

  または、週次更新（run_pipeline.py）の後に手動で実行してください。


================================================================
  よくあるトラブル
================================================================

【ダッシュボードが開かない】
  → ターミナルで「python3 run.py」を実行してから
    ブラウザで「http://localhost:8000」を開いてください。

【「command not found: python3」というエラーが出る】
  → Python3 がインストールされていません。
    https://www.python.org/downloads/ からインストールしてください。

【パイプラインが途中で止まった】
  → ダウンロードが多い場合は時間がかかります。
    5分以上変化がなければ「Control + C」で止めて、
    「python3 pipeline/run_pipeline.py --skip-download」を試してください。

【全銘柄が「見送り」になる】
  → 相場全体が下落トレンドの時は正常な動作です。
    「市場ヘルス」セクションで全体スコアを確認してください。
    スコアが30%未満の場合は「Bearish（弱気相場）」です。
